一日一回hosts2chのhostsデータをダウンロードしてミラーリングする。  
コミットメッセージはブログRSSの先頭itemを拾ってセットしている。このためミラーリングとブログ(RSS)更新のタイミングによっては古い情報を拾ってくる可能性がある。
